Após gravar o respJS_.html, carregue-o em um navegador.

Alguns navegadores acessam o console com F12, outros pela opção de inspecionar elementos. É necessário selecionar a aba do console no quadro que aparecerá.

JS: Bootstrap

O Bootstrap 4 é a versão mais recente do Bootstrap, que é o framework mais popular de HTML, CSS e JavaScript para o desenvolvimento de websites responsivos e para disposivos móveis.

Bootstrap 4 é totalmente gratuito para baixar e usar.

Vantagens do Bootstrap:

  • Fácil de usar: qualquer pessoa com conhecimento básico de HTML e CSS pode começar a usar o Bootstrap
  • Recursos responsivos: o CSS responsivo do Bootstrap se ajusta a telefones, tablets e desktops
  • Primeira abordagem para dispositivos móveis: no Bootstrap, os estilos para dispositivos móveis fazem parte da estrutura principal
  • Compatibilidade do navegador: o Bootstrap 4 é compatível com todos os navegadores modernos (Chrome, Firefox, Internet Explorer 10+, Edge, Safari e Opera)

Para usar o Bootstrap4 em seu site, basta incluir um link de uma CDN (Rede de entrega de conteúdo, sigla em inglês) nas suas páginas da web (aqui, da MaxCDN):
